Formato de Texto en CSS

Curso de Universidad CSS donde aprenderás a crear aplicaciones Web con diseños increíbles en el menor tiempo posible.

Un texto es un conjunto de enunciados que transmiten un mensaje ordenado y coherente, cuya extensión puede variar. Existen diferentes tipos de textos de acuerdo a su estructura y función comunicativa: textos literarios, textos científicos, textos explicativos, textos persuasivos, textos argumentativos.
